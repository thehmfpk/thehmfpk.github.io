# The Plan to Keep Building

## How I Will Add My Next Case Study

Every new project will be added to my portfolio using the same three-part case study structure:

### 1. Problem
Explain the real-world problem, challenge, or need that the project solves.

### 2. What I Did
Describe the solution I built, including:
- Technologies used
- AI tools/models/workflows used
- Development process
- Key features implemented

### 3. What Came Of It
Document the results:
- Project demo
- GitHub repository
- Screenshots
- Learnings
- Improvements and future plans

After completing a project, I will:
1. Upload the code to GitHub.
2. Write proper project documentation.
3. Deploy the project if possible.
4. Add a new case study section to my portfolio website.
5. Share the project update on LinkedIn.
