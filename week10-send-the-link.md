## How I Will Add My Next Case Study

Every new project I complete will follow the same three-part structure:

1. **Problem** – What challenge or opportunity did the project address?
2. **What I Built** – The AI tools, technologies, and workflow I used.
3. **Results** – The outcome, lessons learned, metrics, screenshots, demo, and GitHub repository.

After completing a project, I will:

* Push the source code to GitHub.
* Update the README with documentation.
* Deploy the project if applicable.
* Add a new case study to my portfolio website.
* Share the update on LinkedIn.
